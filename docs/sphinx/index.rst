SciTeX Etc
==========

**scitex-etc** provides miscellaneous utilities for the SciTeX ecosystem:
parameter-grid iteration (``count_grids`` / ``yield_grids``), substring/regex
search (``search``), and chat-pane / terminal / markdown media-reference
detection & display (``media.render``).
